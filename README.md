Customer Behavior Analysis 
Overview

This project focuses on analyzing customer shopping behavior to understand purchasing patterns, spending habits, and customer preferences. The analysis involves loading and exploring a dataset, cleaning and transforming the data, performing SQL-based analysis, and creating visual insights through a dashboard. The goal is to generate meaningful insights that help businesses make data-driven decisions.

Dataset

The dataset contains information related to customer shopping activities such as:

Customer ID

Age and Gender

Product Category

Purchase Amount

Purchase Frequency

Subscription Status

Payment Method

Discount Applied

This data helps analyze customer demographics, purchasing patterns, and spending trends.

Tools & Technologies

The following tools were used in this project:

Python – Data loading and preprocessing

Pandas – Data cleaning and exploratory data analysis

Jupyter Notebook – Running Python analysis

SQL – Querying and analyzing data

PostgreSQL / MySQL / Microsoft SQL Server – Database analysis

Microsoft Power BI Desktop – Data visualization and dashboard creation

Gamma (AI presentation tool) – Presentation creation

Project Steps
1. Data Loading

Loaded the dataset using Python.

Inspected dataset structure and column information.

2. Exploratory Data Analysis (EDA)

Checked data types and summary statistics.

Identified trends in purchase behavior and spending patterns.

3. Data Cleaning

Handled missing values.

Created new derived columns (e.g., purchase frequency in days).

Standardized data formats.

4. SQL Analysis

Imported data into a database.

Wrote SQL queries to calculate:

Average purchase amount

Total revenue

Customer counts by subscription status

Spending trends

5. Dashboard Creation

Built an interactive dashboard using Power BI to visualize:

Revenue by category

Customer demographics

Purchase frequency trends

Subscription vs non-subscription spending

6. Reporting

Generated a project report summarizing the analysis and insights.

Created a presentation using Gamma to explain the project findings.

Dashboard

The Power BI dashboard provides visual insights into customer behavior including:

Customer distribution by gender and age

Total revenue by product category

Purchase trends by season

Subscription vs non-subscription spending comparison

These visualizations help identify key customer segments and purchasing patterns.

Results & Insights

Key insights from the analysis include:

Identification of high-spending customer groups.

Understanding how subscription status impacts spending behavior.

Detection of frequently purchased product categories.

Insights into customer purchase frequency patterns.

These insights can help businesses optimize marketing strategies and improve customer engagement.
